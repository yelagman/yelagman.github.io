My Work

Multilingual Eyetracking Research Project

A multilingual eye-tracking experiment analyzing how native and non-native speakers read across 10+ languages using Pygaze and EyeLink. Published at ACM ETRA 2023.

VR Self-Motion Simulation

A Unity VR environment built with C# and OpenXR to simulate natural self-motion, investigating human perception in immersive spaces.

Camel Farm Management Mobile App

A Kotlin app built with Jetpack Compose and Firebase to help farmers manage camel herds, track health, and sync data in real time across devices.

International Program Web Dashboard

A Flask-based web app for managing international student workflows. Features a Bootstrap dashboard, SQL backend, and 90%+ PyTested REST API.

Room Portfolio

A 3D interactive portfolio built with Three.js, GSAP, and Blender to showcase projects in a dynamic virtual room environment. Styled with Sass and include-media, this website blends creative visuals with smooth animations and handcrafted 3D models.

Kazakh Predictive Keyboard

An offline keyboard for Kazakh built with SQLite and Hugging Face NLP models.

Numberboxd: TV Tracker App

An Android app that lets users rank and track their favorite TV shows, built in Kotlin with the TMDB API.

TuneTrends: Music Popularity Analysis

An ML project analyzing 6,000+ Billboard tracks using Scikit-learn, SHAP, and Spotify API to predict song popularity trends.
